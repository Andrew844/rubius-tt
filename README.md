# Rubius - тестовое задание
## Установка
Из корневой директории:
<br/>
<code>
  cd client
</code>
<br/>
<code>
  npm install
</code>
<br/>
<code>
  cd ..
</code>
<br/>
<code>
  npm install
</code>

## Запуск
В корневой директории:
<br/>
<code>
  npm run start
</code>
